Splice() an Array

The splice() method can be used to add new items to an array.

For example we have declaration like this...let fruit = ["banana", "orange", "apple"];

We apply the splice() to the fruit array...fruits.splice(2, 0, "lemon", "kiwi");

The first parameter (2) defines the position where the new elements should be added (or spliced in).

The second parameter (0) defines how many elements should be removed

The rest of the parameters('lemon', 'kiwi') define the new elements to be added

The splice() method returns the array with the deleted items
